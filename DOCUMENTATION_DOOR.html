🔑Door Project

Zero-Touch Building Access Solution via Wear OS

📋 Overview

The Door Project is a bridge automation implementation that connects Wear OS devices to Android smartphones to streamline the use of HID Mobile Access technology. The solution eliminates the need for manual interactions with the phone to unlock the screen before approaching a door reader with the watch.

WATCH MODULE (:app)

The Stealth Trigger

Executes a "silent" operation without a visible user interface.

  • Bluetooth Guard: Active connection state validation.
  • Dynamic HID Search: Automatic package location using contains("hid").
  • Translucent Bridge: 100% transparent activity.
MOBILE MODULE (:mobile)

The Secure Bridge

Handles system permissions and the device security bypass.

  • WakeLock: Remotely wakes up the screen hardware.
  • Keyguard Dismiss: Requests lock screen dismissal via official API.
  • Stealth Mode: Auto-termination and hiding from recent apps.

🚀 Operation Flow

1
Activation: User taps the app icon on the watch.
2
Handshake: Watch sends /unlock_phone signal via Bluetooth.
3
Unlock: Phone wakes up and dismisses lock screen (if Smart Lock is active).
4
Transition: Watch immediately opens the HID Mobile Access app.

🛡️ Security Requirements

For optimal performance, the following is recommended:

  • Smart Lock: Configure the watch as a "Trusted Device" on the phone.
  • Permissions: Grant QUERY_ALL_PACKAGES on the watch to detect the HID app.
  • Bluetooth: Keep connection active (the app alerts if disabled).
